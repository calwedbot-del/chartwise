- **Data:** CoinGecko API (crypto), Yahoo Finance (stocks)

## Getting Started

```bash
# Clone the repo
git clone https://github.com/DoubleO7Rintu/chartwise.git
cd chartwise

# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```
